# Titan Scheduler

An application to help generate class schedules for students.

## Description

This program utilizes multiple API's to help instantly generate a student's schedule for the next semester. It takes in their TDA (Titan Degree Audit) and loads the best fit. 

## Getting Started

### Dependencies

* FastAPI
* uvicorn
* pydantic
* python-multipart
* openai
* selenium
* pandas
* pyyaml
* sqlalchemy

### Instructions

* CD into Backend/TitanAPI/app directory
* Ensure all dependancies are installed
* Enter: 
`uvicorn main:app --reload`
* API should be hosted on local host
* Go to Titanscheduler.org
* download and input the test tda named "TDA_Example" from github
